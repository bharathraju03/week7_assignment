# Week 7 - Data Cleaning and Exploration using Pandas

## Objective

Understand data loading, exploration, cleaning, and transformation using Pandas.

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Dataset

* Sample - Superstore.csv

## Tasks Performed

### Step 1: Load Dataset

Loaded the Superstore dataset into a Pandas DataFrame.

### Step 2: Explore Data

Used:

* head()
* tail()
* shape
* columns
* dtypes

### Step 3: Handle Missing Values

* Identified missing values using isnull()
* Filled missing values using fillna()

### Step 4: Basic Operations

* Filtered rows where Sales > 1000
* Selected Product Name and Sales columns

### Step 5: Remove Duplicates

Removed duplicate records using drop_duplicates().

### Step 6: Create Derived Column

Created a new column:

total_amount = Sales × Quantity

### Step 7: Save Cleaned Dataset

Saved processed data into:

cleaned_superstore.csv

## Key Insights

1. Pandas provides efficient tools for data analysis and preprocessing.
2. Data exploration helps understand dataset structure.
3. Missing value handling improves data quality.
4. Filtering and column selection enable focused analysis.
5. Removing duplicates prevents redundant records.
6. Derived columns help generate additional business insights.
